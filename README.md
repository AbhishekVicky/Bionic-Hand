# 🦾 Bionic Hand

A 3D-printed robotic hand built using Arduino and servo motors that mimics basic human finger movements. This project demonstrates embedded systems, robotics, gesture control, and prosthetic technology concepts.

---

## 📖 Project Overview

The **Bionic Hand** is a humanoid robotic hand designed using modular 3D-printed parts and servo motors. The project focuses on replicating basic human hand movements through programmed servo control. It was assembled by understanding each mechanical component, integrating electronics, and programming the motion using Arduino.

This project helped me improve my practical skills in robotics assembly, servo motor control, embedded programming, and hardware debugging.

## ⚙ Working Principle

The Arduino Nano controls multiple servo motors that drive the movement of individual fingers. Each servo rotates to predefined angles, allowing the robotic hand to imitate simple human hand gestures. The modular design makes it easy to repair, modify, and upgrade.
